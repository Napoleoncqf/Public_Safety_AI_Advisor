# 索尔维微笑挑战赛 | 行动代号：“首秀”

## 1. 核心概念

一场由AI担任随机裁判的、顶尖科学家间的“表情包”淘汰赛。

- **赛制**: “诺贝尔冠军之路”。16位诺贝尔奖得主进行单败淘汰赛，决出总冠军，该冠军将获得挑战“最终隐藏Boss”爱因斯坦的资格。
- **核心技术**: ComfyUI图生图工作流，配合“一人怒一人笑”的提示词，实现随机裁决。
